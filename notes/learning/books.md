# Linux & Systems Programming Reading List

## Foundations

- **The Linux Programming Interface** - Michael Kerrisk
  System calls, processes, signals, threads, IPC, sockets, file I/O

- **How Linux Works, 3rd Edition** - Brian Ward
  Boot process, devices, networking, shell, filesystems

- **UNIX and Linux System Administration Handbook, 5th Edition** - Nemeth, Snyder, Hein, Whaley, Mackin
  User management, networking, security, deployment

- **Linux From Scratch** - Gerard Beekmans
  Build a Linux system from source (free: https://www.linuxfromscratch.org/)

## Networking

- **TCP/IP Illustrated, Volume 1** - W. Richard Stevens
  TCP/IP protocols, packet analysis

- **Computer Networking: A Top-Down Approach, 8th Edition** - Kurose, Ross
  Application to physical layer, structured textbook format

- **Beej's Guide to Network Programming** - Brian Hall
  Socket programming in C (free: https://beej.us/guide/bgnet/)

## Kernel Development

- **Linux Kernel Development, 3rd Edition** - Robert Love
  Process management, scheduling, memory management, VFS, block I/O

- **Understanding the Linux Kernel, 3rd Edition** - Bovet, Cesati
  Detailed kernel internals reference

- **Linux Device Drivers, 3rd Edition** - Corbet, Rubini, Kroah-Hartman
  Kernel modules, device drivers (free: https://lwn.net/Kernel/LDD3/)
